📌 Project Description

HabitTrack is a full-stack database-driven web application designed to help users create, manage, and track their daily habits while gaining insights into their progress over time.

The application allows users to log habit activity, organize habits by category, and view detailed progress reports. It leverages structured relational data and advanced SQL queries to support dynamic data retrieval and analysis.

On the backend, the system is built using Node.js and Express, with an Oracle SQL database handling data storage and query execution. The application implements a range of database operations, including selection, projection, joins, aggregations (GROUP BY, HAVING), nested queries, and division queries to support complex analytical features.

The project emphasizes efficient data modeling, query optimization, and backend API design, demonstrating how relational databases can be used to power real-world applications with meaningful analytics and reporting functionality.
